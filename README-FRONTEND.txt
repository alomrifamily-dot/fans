Fans Frontend (static)
======================
ملفات واجهة بسيطة تتصل بباكند Fans Wallet.

ملفات مهمة:
- index.html: الصفحة الرئيسية
- claim.html: واجهة المحفظة (منح/استبدال/سحب/تحويل + عرض الرصيد والسجل)
- leaderboard.html: واجهة متصدرين تجريبية
- assets/config.js: عدّل API_BASE إذا تغيّر رابط الباكند

نشر على GitHub Pages:
- ادفع هذه الملفات إلى مستودع alomrifamily-dot/fans
- من Settings → Pages → Branch: main و Folder: /(root)
